# Rick and Morty API

Single-page web app that fetches and displays random characters from the [Rick and Morty API](https://rickandmortyapi.com/).

## Features

- Random character generation with one click
- Displays character name and sprite image
- Error handling for invalid character IDs
- Responsive, centered layout with background image

## Tech Stack

- HTML5
- CSS3 (Roboto font, gradient styling)
- Vanilla JavaScript (Fetch API, async/await)

## Usage

Open `index.html` in a browser or visit the [live demo](https://tomas-barros1.github.io/Rick-and-Morty-API/).

No build step required.
